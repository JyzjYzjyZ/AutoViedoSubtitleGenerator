# AutoViedoSubtitleGenerator
用于自动为视频添加字幕
## introduction
目前正处于测试阶段

调用ASR/STT  huggingface-facebookwave;vosk;mt5-zh-en-ja

支持config 全离线的字幕生成

必要库见create/.bat
代码完善后放出编译版本，当前可通过menu.spec快速编译
